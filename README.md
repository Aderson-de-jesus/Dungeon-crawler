# Dungeon-crawler
# nome

Need of a Dungeon

# Desenvolvedores

Aderso de jesus

# História do Jogo

Em Need of a Dungeon, o jogador e um fanatico por dungeons, ele adora explorar dungeons e sempre procura uma dungeon para explorar

# Objetivo

Chegar ao final de cada fase, sobrevivendo aos perigos, resolvendo obstáculos e encontrando a saída para a próxima dungeon.

# Controles

W	Move para cima e muda o símbolo para ^

A	Move para a esquerda e muda o símbolo para <

S	Move para baixo e muda o símbolo para v

D	Move para a direita e muda o símbolo para >

I	Interage com o objeto à frente;

O	Ataca a célula à frente;

Elementos do Jogo;

Símbolo	Significado;

<	Jogador virado para a esquerda;

^	Jogador virado para cima

(>)	Jogador virado para a direita

v	Jogador virado para baixo

(*)	Parede (bloqueia passagem)
  
(#)	Espinho (morte ao pisar)

k	Caixa (bloqueia passagem, pode ser destruída)

O	Botão (ativa ações quando pressionado)

D	Porta fechada (bloqueia passagem)

@	Chave (abre portas fechadas ao interagir)

=	Porta aberta (libera passagem)

L	Escada (leva à próxima fase)

X	Monstro Tipo 1

Y	Monstro Tipo 2

Z	Boss Final
